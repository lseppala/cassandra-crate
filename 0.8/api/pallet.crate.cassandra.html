pallet.crate.cassandra documentation

Pallet crate to install, configure and operate Cassandra.

See Also

http://wiki.apache.org/cassandra/Operations.

cassandra-config-changed-flag

var

Flag for recognising changes to configuration

[{:keys [main ram-fraction], :or {ram-fraction 0.4}, :as settings}]

Update settings with cluster specific values.

[{:keys [owner group config-dir], :as settings} filename file-source]

Helper to write config files

configure

fn

[{:keys [instance-id], :as options}]

Configure cassandra

cron-time

fn

[index n]

Return a vector with a minute and an hour at which to perform an operation. Assumes an operation should be run once a day, and tries to spread the operation for index'th of n operations over the course of a Sunday.

cassandra-env.sh file content.

initial-token

fn

[supplied-tokens token-index]

Compute node token. If supplied-tokens is nil, then tokens are equi-spaced over the token-space based on the number of nodes in the group.

[]

repair-cronjob-cron-file

fn

[{:keys [keyspace day], :or {day 0, keyspace ""}}]

Run repair on a node in a group, staggering when the repair is run. The day of the week to run the repair can be chosed with day.

Returns a server spec to install and configure a cassandra server spec.

Run the cassandra service.

settings

fn

[{:keys [server], :as settings} {:keys [instance-id], :as options}]

Settings for cassandra

:server : a map of options as specified in http://www.datastax.com/docs/0.8/configuration/node_configuration. These will be serialised to yaml.

token-space

var

Maximum of the token space

tokens

fn

[count]

Return tokens for a cassandra cluster of size count, evenly distributed over token-space. See http://wiki.apache.org/cassandra/Operations#Token_selection.

topology-properties

fn

[group-data-centers]

cassandra-topology.properties content. group-data-centers is a map from group name to a vector with data-center and rack names. This implies a group is rack specific. See http://www.datastax.com/docs/0.8/clusterarchitecture/clusterplanning#configuring-the-propertyfilesnitch
